# CV And Gate Ranges

### All CV inputs respond to -5 V…+5 V. Every knob is treated as an offset: with the knob at 50%, a -5 V signal will turn it all the way left, and a +5 V signal will turn it all the way right. All logic inputs outputs are 0-5v. All Quantized outputs are scaled for 1v/octave pitch tracking.
